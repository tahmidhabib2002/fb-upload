System User Token (Recommended)

For long-term stability, use a System User Token from Facebook Business Manager so your app doesn't lose access when your personal session expires.

  1. Go to Business Manager → System Users
  2. Create a System User with Employee role
  3. Add your Pages to the System User with Advertiser or Admin access
  4. Generate a System User token with the required permissions
  5. Paste that token in the field above instead of a personal token
